Place Limine BIOS files here:
  - limine-bios-cd.bin
  - limine-bios.sys
  - limine.exe

Optional (UEFI boot support):
  - limine-uefi-cd.bin
  - BOOTX64.EFI (and/or BOOTIA32.EFI for 32-bit UEFI)

Download the Limine binary release from the official GitHub releases page,
extract the files above, and copy them into this folder.
