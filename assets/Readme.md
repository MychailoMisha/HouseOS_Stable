# Assets Folder Guide

This folder is used for custom user assets.

You can add your own files to customize the system appearance.

Required file names:

* background.png — used as desktop background
* cursor.png — used as mouse cursor

Instructions:

1. Place your image files inside this folder.
2. Make sure the file names are exactly:

   * background.png
   * cursor.png
3. Restart the system (or reload graphics) to apply changes.

Notes:

* Supported format: PNG
* Recommended resolution for background: 1920x1080
* Cursor image should be small (e.g. 32x32 or 64x64)

If files are missing, default system assets will be used.
